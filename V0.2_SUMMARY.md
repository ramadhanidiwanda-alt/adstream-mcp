## 🚀 Usage Examples

### Custom Rules (Phase 1)
```typescript
import { RuleEngine } from 'meta-ads-agent-skill';

const customRule = {
  id: 'my-rule',
  name: 'High Spend Alert',
  conditions: [{ metric: 'spend', operator: '>', value: 100000 }],
  logic: 'AND',
  action: 'Review budget',
  priority: 'high',
  enabled: true,
};

const engine = new RuleEngine();
const results = engine.applyRulesToInsights(insights, [customRule]);
```

### Pre-built Templates (Phase 2)
```typescript
import { ecommerceRules, RuleEngine } from 'meta-ads-agent-skill';

const engine = new RuleEngine();
const results = engine.applyRulesToInsights(insights, ecommerceRules);
```

### All Templates Combined
```typescript
import { allRuleTemplates, RuleEngine } from 'meta-ads-agent-skill';

const engine = new RuleEngine();
const results = engine.applyRulesToInsights(insights, allRuleTemplates);
```
